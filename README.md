### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:shegy28/crypto-metrics-Webapp.git
```
### Install

To run on local machine, open your terminal and run the following:

```sh
  npm install
  npm run build 
  npm start
```
